How to add favicon to your game:

  1. Right-click the image above and save it as "favicon.ico" or "favicon.png"
  2. Place the file in the same folder as index.html
  3. Add this line to the <head> section of index.html:
    <link rel="icon" href="favicon.ico" type="image/x-icon">
